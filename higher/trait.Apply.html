Expand description

Apply takes an Apply<Fn(A) -> B> (or, rather, an Apply<ApplyFn<'a, A, B>> specifically) and applies it to an Apply<A> to produce an Apply<B>.

In simpler terms, it takes a value of type A inside some context (“some context” here being the type which implements Apply) and a function Fn(A) ->B inside a similar context and produces a similar context containing the result (B) of applying the function to the value. Some concrete examples of “some context” are Option, Vec and futures.
